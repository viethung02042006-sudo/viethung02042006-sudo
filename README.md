## 📂 Featured Projects

### 🎬 Movie Ticket Booking System
*A full-stack, enterprise-ready ticket booking and cinema management application.*
* **Tech Stack:** Spring Boot 3, React 19, TypeScript, Tailwind CSS v4, PostgreSQL.
* **Key Features:**
  * Implemented database transactions under strict isolation levels to prevent double-booking.
  * Solved concurrent seat reservation conflicts using **Pessimistic Write Locking (`SELECT FOR UPDATE`)**.
  * Designed a custom Many-to-Many entity mapping mechanism in JPA to bypass default Hibernate performance issues and support future relationship metadata extension.
  * Built a dynamic, stateful seat map React component displaying real-time booking statuses.

### 🥊 Boxing Game
*An interactive game emphasizing fluid gameplay mechanics and physics simulations.*
* **Key Features:**
  * Structured utilizing a robust game loop pattern, decoupling rendering rate from update mechanics.
  * Developed custom state machines (FSM) to handle boxing character animations, moves, and action combinations.
  * Implemented responsive keyboard controls and precise collision detection algorithms.

### 💼 AIMS (Analysis, Design & Implementation of Software Systems)

* **Key Features:**
